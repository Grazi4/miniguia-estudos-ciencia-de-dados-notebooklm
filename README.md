# 📊 Guia de Estudos em Ciência de Dados com NotebookLM

Este projeto tem como objetivo a criação de um guia de estudos em Ciência de Dados utilizando a ferramenta **NotebookLM**, com foco na organização, síntese e estruturação do conhecimento.

---

## 🧠 Objetivo

Desenvolver um material estruturado que auxilie no aprendizado de Ciência de Dados, utilizando inteligência artificial para:

* Organizar conteúdos
* Gerar resumos
* Apoiar a compreensão dos temas estudados

---

## 🛠️ Ferramenta utilizada

O projeto foi desenvolvido com o apoio do **NotebookLM**, uma ferramenta baseada em inteligência artificial que permite:

* Analisar documentos
* Gerar resumos automáticos
* Criar conexões entre conteúdos
* Apoiar o estudo de forma guiada

---

## 🗂️ Estrutura do guia

O guia de estudos está organizado nas seguintes áreas:

### 💻 Programação

* **Python**
* Lógica de programação

### 🗄️ Banco de Dados

* SQL
* Modelagem de dados

### 📊 Análise de Dados

* Limpeza de dados
* Análise exploratória
* Visualização

### 📈 Business Intelligence

* Power BI
* Criação de dashboards

### 🤖 Machine Learning

* Conceitos básicos
* Principais algoritmos

---

## 📚 Metodologia

O desenvolvimento do guia seguiu as etapas:

1. Seleção de conteúdos relevantes
2. Inserção dos materiais no NotebookLM
3. Geração de resumos e sínteses
4. Organização em trilhas de estudo

---

## 🎯 Resultado esperado

Criar um material que:

* Facilite o aprendizado de Ciência de Dados
* Estruture o conteúdo de forma progressiva
* Utilize IA como apoio educacional

---

## 📌 Contexto acadêmico

Este projeto foi desenvolvido como parte de um curso, com o objetivo de aplicar ferramentas de inteligência artificial no processo de aprendizagem.
